# Element.remove()

The `Element.remove()` method ==removes the element (or node) from the DOM==.

## Syntax

```js
element.remove();

// or
node.remove();
```

## Parameters

none

## Examples

```js
const element = document.getElementById('demo');
element.remove();
```
