# Array.prototype.pop()

The `pop()` method ==removes== the ==last item== from an array and returns that item. The `pop()` method changes the original array.

## Syntax

```js
array.pop();
```

## Parameters

None

## Return Value

The removed item from the array; `undefined` if the array is empty.
